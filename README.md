### Sky planet Shop online

- Change slides in carousel
- Add product to Cart
- Add product to favourites
- Check the Cart
- Remove from Cart
- Change pages

#### Technologies used:

- HTML5

- JavaScript
- Ajax
- Php (in cooperation with a backend programmer)
